De zoekindex en bijhorende API maakt het mogelijk om openbaar gemaakte overheidsdocumenten te doorzoeken in het kader van de Wet Open Overheid (WOO).

Registratiecomponenten bieden publicaties aan om ze te indexeren (of in te trekken).

The search index and accompanying API make it possible to search through published government documents in the context of the "Open Government Law", for which the Dutch abbreviation is "WOO".

Registration components send publications to this search component to initiate indexing (or to retract earlier indexed data).
